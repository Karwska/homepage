TYDZIEŃ III

Jeśli chodzi o html, to rozwijaliśmy dalszą wiedzę na temat semantyki. Poznałam znaczniki, br, b, em, strong, i - a także równicę między nimi.
Bardzo istotną rzeczą była nauka dołączania obrazków i linków. Dzieki temu powstała też nawigacja na tej stronie :)

Jeśli chodzi o CSS to poznałam zasady specyficzności CSS oraz to, żeby ZAWSZE stylować po klasach! CSS z tygodnia 3 na początku wydawał się wymagający, ale później przyszedł tydzień 4 i zburzył mój światopogląd...

TYDZIEŃ IV

Jak na razie najbardziej intensywny tydzień. Poznałam zasady pracy z GITEM, baaaardzo dużo CSS (żyje od dzisiaj w pudełku, a nie marzeniami). Poszerzyłam też wiedzę z JS, natomiast jeszcz jego zastosowanie owiane jest mgłą tajemnicy. Co najważniejsze - commituje teraz jak zła.
